# サキ 包分析与修复说明

## 发现的差分规律

在 `サキＡ_0~3.json` 中：

- 第 1 项是画布大小，例如 `4385 x 5500`
- 顶层组有两个：
  - `私服`
  - `差分`
- `私服` 组下只有 1 张身体底图，例如 `正面 (layer_id=517)`
- `差分` 组下是脸部差分：
  - 普通表情：`恍惚 / すまし / 驚き / 呆れ / 寂しい / 怒り / 真剣 / 薄ら笑い / にっこり / 微笑`
  - 红晕：`頬常時 / 頬照れ`

最终合成规则为：

`1 张身体底图 + 0/1 张表情 + 0/1 张红晕`

并按 JSON 中每层的 `left/top` 贴到透明画布上。

## 之前程序出错的原因

旧版把“只有顶层组名等于 `表情` 的组”才当成表情组。

但 `サキ` 这套资源里，脸部差分组名称是 **`差分`**，不是 `表情`。

结果导致：

- `差分` 组被误判成“身体/服装组”
- 身体下拉框里出现了 `差分/恍惚` 这种错误选项
- 默认身体层也被错误选成了表情图层
- 预览里就只剩一张脸，没有身体

## 这次修复

1. 增加了对 `差分` 组的识别，作为表情组处理
2. 增加了更稳的启发式判断：
   - 名称包含表情差分语义
   - 子图数量多且面积小
   - 子图集中在脸部区域
   - 存在 `頬*` 图层
3. 默认身体层改为：
   - 优先选“可见且面积最大的身体层”
   - 避免把脸部差分选成默认身体

## 修复后效果

对 `サキ.zip` 的正确分类应为：

- 身体/服装：`私服 / 正面`
- 表情：10 种
- 红晕：2 种

默认组合：

- 身体：`私服 / 正面`
- 表情：`微笑`
- 红晕：`頬常時`
