结论
- JSON 第 1 个对象是画布尺寸。
- JSON 其余对象里，顶层 group_layer_id = null 的项是“分类组”，例如 裸、制服１、表情。
- 非表情组的子图层就是身体/服装底图；每个子图层的 layer_id 对应一个同名 TLG 文件后缀。
- 表情组的子图层里，名称以“頬”开头的可视为红晕层，其余为表情层。
- 最终图 = 1 张身体底图 + 0/1 张表情层 + 0/1 张红晕层，按 JSON/SINFO 顺序和 left/top 坐标透明叠加。
- TLG 文件头里的宽高与 JSON 中同 layer_id 的 width/height 完全一致，可用于校验。
